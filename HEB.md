Key Hebrew NLP datasets focusing on root words (shoresh), morphology, and lemmatization include the Hebrew Treebank, the Verb Complements Lexicon, and specialized tools like the Dikduk morphological analyzer. These resources, often curated by groups like NNLP-IL, provide annotated data for root extraction, segmentation, and semantic analysis. 
Key Datasets and Resources

Hebrew Treebank 2.0: Contains 6,500 hand-annotated sentences with full word segmentation, offering morphological data suitable for analysis.
Verb Complements Lexicon: Includes a verb_root column and is a key source for identifying verb roots.
Dicta/Nakdimon: A deep-learning-based tool for vocalization that implies high-level morphological parsing for roots.
